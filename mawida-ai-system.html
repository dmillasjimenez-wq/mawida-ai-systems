❓ Cómo usar Mawida AI

1. Cargá tu biblioteca — Ve a Biblioteca de Marca y subí tus manuales PDF, notas de tono, logos. Esto hace que la IA respete tu identidad.

2. Chateá con tu equipo — Desde el Home elegí un agente y conversá. Activa "Usar contexto de marca" para que use tu biblioteca.

3. Brief Automático — Dale un objetivo y el equipo completo genera la campaña en cadena. El feature más potente.

4. Features Pro — Auditor de Marca, Asistente Legal, Generador Visual y Narrador en el menú lateral.

5. Compartir — Exportá tu biblioteca como .json y pasásela a tu equipo. Ellos la importan y tienen tu setup completo.

📤 Compartir el sistema

🌐 Opción A — Compartir el archivo

Pasá el archivo HTML por WhatsApp, Drive o mail. Cada persona necesita su propia API Key.

📦 Opción B — Exportar biblioteca

Exportá toda tu configuración y manuales. Quien lo importe tiene tu setup completo.
